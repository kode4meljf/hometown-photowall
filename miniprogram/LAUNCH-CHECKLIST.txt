【老根茶村 · 上线前公众平台手动清单】

1. 用户隐私保护指引
   路径：微信公众平台 → 设置 → 服务内容声明 → 用户隐私保护指引
   勾选/填写与代码一致：
   - 微信登录标识（OpenID）
   - 昵称、头像、性别/地区/简介（用户主动填写）
   - 相册、相机（发帖选图、编辑头像）
   - 保存到相册（大图下载）
   - 用户发布的内容（图片、标题、描述、评论）
   不要勾选：GPS 精确定位（本小程序仅手动填写地点标签）

   代码侧：app.json 已设置 __usePrivacyCheck__: true
   公众平台指引须与上述权限一致，否则真机选图/登录会异常。

2. 服务类目
   按实际功能选择；UGC 类需在审核补充说明中写明：
   - 微信内容安全 API（文本/图片）
   - 作品与评论举报入口（详情页、设置页反馈）
   - 后台 adminApi 人工审核与举报处理

3. 云开发
   - 创建数据库集合：users、posts、post_comments、feedbacks
   - 部署云函数：auth、posts、feedback、stats、signin、adminApi
     （均需「上传并部署：云端安装依赖」，确认各函数内 common/ 子目录已上传）
   - adminApi 生产环境变量：
     ADMIN_TOKEN_SECRET（必填）
     ADMIN_CORS_ORIGINS（Web 管理后台域名，逗号分隔）
   - 检查数据库/存储安全规则，禁止客户端越权读写

4. Web 管理后台（可选，运营用）
   - 仓库 web/ 目录，npm run dev 本地调试
   - 配置 web/.env：VITE_CLOUDBASE_HTTP_BASE、VITE_ADMIN_API_PATH
   - HTTP 访问服务须配置 /adminApi 路由，默认域名续期有效

5. 提审补充说明（建议复制）
   - 微信授权即可登录，无需额外测试账号
   - 发帖/评论/点赞/举报/意见反馈/注销均已实现
   - 不涉及支付、直播

6. 真机自测（__usePrivacyCheck__ 已开启）
   - 未同意隐私：选图/保存/登录应引导同意
   - 登录弹窗可打开协议与隐私政策
   - 评论删除、评论举报、作品举报
   - 设置页提交意见反馈
   - 注销账号后数据不可恢复
   - 详情页分享：生成海报（含小程序码）保存相册；需部署 posts 云函数 getShareQrCode
   - 积分栏点击无「开发中」提示

7. 代码改动后必做
   - 重新上传并部署云函数：auth、posts、feedback、adminApi（若改动了对应逻辑）
   - 开发者工具上传小程序代码 → 提交审核
